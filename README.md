# 防治排班系統 (Pest Control Scheduling System)

這是一個基於 HTML5/JavaScript 的單頁 Web 應用程式 (SPA)，專為防治服務業設計，用於管理每月行程、偵測衝堂、視覺化排班以及產出機器人發送清單。

## 📋 功能分類介紹

### 1. 核心排班與資料管理
*   **Excel 智慧匯入**：支援匯入 `.xlsx` 格式的排班表，自動對齊客戶名稱、日期、時間與防治師。
*   **資料持久化**：資料按月份儲存在瀏覽器 `localStorage` 中，重新整理網頁資料不遺失。
*   **自動儲存與復原**：任何更動皆自動儲存，並支援 30 步的操作復原 (`Ctrl+Z`)。
*   **衝堂偵測**：自動標示同一防治師在時間重疊的案場，預防排班衝突。

### 2. 視覺化維護介面
*   **📋 排班明細**：提供強大的搜尋與篩選功能（按防治師、區域、類型、狀態），並支援直接編輯日期。
*   **📅 DayView (日進度)**：
    *   **拖曳排班**：在時間軸上直接移動區塊來修改時間或變更防治師。
    *   **休假標示**：輸入防治師休假日期，系統會即時標註並偵測衝突。
*   **📆 週覽**：以週為單位（一至五）快速瀏覽每日防治能量分配。

### 3. 自動化與外掛支援
*   **🤖 機器人清單 (Bot List)**：自動將客戶依特性分類為第一批（固定）與第二批（例行），方便自動化機器人發送通知。
*   **📤 多元匯出**：
    *   **筋斗雲 Excel**：產生符合平台格式的檔案，可直接上傳回系統。
    *   **TSV 字串**：提供機器人清單與行程明細的文字版，方便快速複製。

### 4. 統計與輔助工具
*   **📊 工作量統計**：以圖像化卡片顯示每位防治師本月的總場次。
*   **👥 客戶資料庫**：內建客戶基本資料表（區域、時長、工單需求等）供查詢參考。

## 🚀 使用說明
1.  **匯入**：點擊右上角「📥 匯入 Excel 排班」選擇本月行程表。
2.  **調整**：使用「排班明細」或「DayView」微調時間或防治師分配。
3.  **檢查**：確認統計列中的「⚠️ 衝堂」數為 0。
4.  **匯出**：切換至「匯出」標籤下載 Excel 或複製機器人所需清單。

## 🛠 技術規格
*   **核心**：HTML5 / Vanilla CSS / Vanilla JavaScript (ES6+)
*   **解析庫**：[SheetJS (XLSX)](https://github.com/SheetJS/sheetjs)
*   **效能**：採 DOM Fragment 渲染技術，支援數百筆資料流暢操作。
